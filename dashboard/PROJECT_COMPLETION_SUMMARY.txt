## 🐛 Troubleshooting Quick Reference

| Problem | Solution |
|---------|----------|
| **"could not connect to server"** | Check PostgreSQL is running and .env credentials are correct |
| **"ModuleNotFoundError"** | Run `pip install -r requirements.txt` in correct venv |
| **"Port 8501 already in use"** | Run `streamlit run app.py --server.port 8502` |
| **"No data showing"** | Verify ETL pipeline completed, check database has tables |
| **"Slow loading"** | Clear cache: `streamlit cache clear` |
