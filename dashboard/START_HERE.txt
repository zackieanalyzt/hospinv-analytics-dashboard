## 🐛 Troubleshooting

### "Database Connection Failed"
```bash
# Check PostgreSQL is running
psql -h localhost -U postgres

# If command not found, PostgreSQL is not installed or not in PATH
# Check .env file has correct credentials
```

### "ModuleNotFoundError"
```bash
# Reinstall dependencies
cd dashboard
pip install -r requirements.txt
```

### "Port 8501 in Use"
```bash
# Use different port
streamlit run app.py --server.port 8502
```

### "No Data Showing"
1. Check Dashboard for data freshness indicator
2. Verify ETL pipeline has completed
3. Confirm database has data: 
   ```bash
   psql -h localhost -U postgres -d analytics -c "SELECT COUNT(*) FROM core.dim_item;"
   ```
