📋 INSTALLATION OPTIONS
────────────────────────────────────────────────────────────────────────────────

OPTION 1: ONE-CLICK START (Easiest)
   Windows: Double-click dashboard\run.bat
   Mac/Linux: bash dashboard/run.sh
   Result: Dashboard opens automatically at http://localhost:8501

OPTION 2: DOCKER (Production-Ready)
   docker-compose up -d
   Result: Full stack running (dashboard + PostgreSQL)

OPTION 3: INTERACTIVE SETUP
   cd dashboard
   python setup.py
   Result: Interactive wizard guides you through setup

OPTION 4: MANUAL SETUP
   cd dashboard
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env
   # Edit .env with database credentials
   streamlit run app.py
